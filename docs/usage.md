# Usage


```sh
$ cd PATH_TO_REPO
$ source venv/bin/activate
```


## View available make commands

```bash
$ make help
```


# Format

```sh
$ make fmt-fix
```


## Lint

```sh
$ make lint
```

## Check types


```sh
$ make typecheck
```

## Test

```sh
$ make test
```


## Run

Run application.

```sh
$ make run
```
